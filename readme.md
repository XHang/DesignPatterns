## 状态模式
什么是状态模式：  
该模式是以面向对象的方式实现状态机的软件设计模式，利用状态模式，通过将每个单独状态实现为状态接口的子类，并调用状态子类的方法来实现状态转换。  

什么时候可以使用状态模式？  
1. 一个对象的行为只取决于一个或者多个动态变化的属性，这样的属性叫做状态    
2. 程序内部包含许多的分支结构，且这些分支依赖于对象状态的变化
